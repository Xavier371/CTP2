Catch the Point

Try to catch the red point

How to Play

Offense Mode

  • You are the blue point, try and catch the red point by moving towards it
  • Red will try to maintain distance and find escape routes

Defense Mode

  • You are the blue point, try and evade the red point by maximizing the distance from it
  • Stay connected to multiple paths to avoid being trapped

Game Basics

  • Each move removes one random edge
  • Win by either connecting points or completely separating them
  • Press Enter to restart anytime
